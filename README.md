## рџљЂ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) в‰Ґ 18
- [Rust](https://www.rust-lang.org/tools/install) + Cargo
- [Tauri CLI](https://tauri.app/start/)
- [llama.cpp](https://github.com/ggerganov/llama.cpp) built and running

### 1. Start llama.cpp server

```bash
# Example with qwen2.5-3b-instruct model
./llama-server \
  -m models/qwen2.5-3b-instruct-q4_k_m.gguf \
  --port 8080 \
  --ctx-size 4096 \
  -ngl 0
```

The app expects the server at `http://localhost:8080`.

### 2. Install JS dependencies

```bash
pnpm install
# or npm install
```

### 3. Run in development

```bash
pnpm tauri dev
```

### 4. Build for production

```bash
pnpm tauri build
```

## рџ§  How AI Content Generation Works

Each game uses a **structured prompt** that instructs the model to respond with **strict JSON only** вЂ” no prose, no markdown fences. This means:

- The small 3B model has a very focused task (fill a template).
- The UI parses the JSON and renders it in a polished interface.
- Malformed output is caught gracefully with retry prompts.

### Prompt files

| File | Output shape |
|------|-------------|
| `lib/prompts/anki.ts` | `AnkiCard[]` |
| `lib/prompts/phraseBuilder.ts` | `PhraseChallenge` |
| `lib/prompts/scenario.ts` | `ScenarioDialogue` |
